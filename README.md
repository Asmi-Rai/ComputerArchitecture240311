# Computer Architecture Labs (CMP 262)

In this repository, you will find **sample VHDL codes** that I learned and practiced during my Bachelor labs.  
These exercises helped me build a foundation in **VHDL basics**, digital logic design, and simulation workflows using **VS Code, GHDL, and GTKWave**.

Each lab folder contains:
- VHDL design files
- Testbenches for simulation
- Waveform outputs
- Lab‑specific README with objectives and explanations

This repo serves as a **summary collection of my learning journey in VHDL programming**.
